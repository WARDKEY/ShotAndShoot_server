## 프로젝트 소개
### 프로젝트명 :  Shot&Shoot

“Shot&Shoot”은 폐기물 사진을 찍으면 종류와 분립배출 방법을 알려주는 앱입니다.
사용자는 커뮤니티에서 폐기물 관련 질문을 공유하고, 가까운 분리수거 대행업체도 확인할 수 있습니다.

***Q. Shot&Shoot의 의미?***

폐기물 사진을 찍고(Shot), 분리수거함에 넣는다(Shoot) 라는 의미로 "Shot&Shoot"이라고 정했습니다.

## 주요 기능

### 폐기물 대행 업체 찾기 기능
- 폐기물 업체 페이지에서 지도에서 사용자 위치를 기반으로 주변의 폐기물 대행 업체 업체 찾기
- 메인 페이지에서 사용자 위치와 가장 가까운 폐기물 대행 업체를 표시
- 특정 폐기물 업체 상세 정보를 확인하고 전화번호 클릭 시 전화걸기

### 커뮤니티 기능
- 제목, 내용, 카테고리를 입력 및 선택하여 게시글 작성
- 게시글 삭제 버튼을 통해 게시글 삭제
- 인기 순으로 게시글을 나타내는 기능
- 게시글 상세화면에 댓글 작성 및 조회
- 댓글 삭제 버튼을 통해 댓글 삭제

### 카테고리 및 검색 필터
- 8개의 카테고리로 게시글 분류
- 게시글 제목, 내용 기반 키워드 검색

### AI 기능
- 게시글 작성 시 AI가 게시물 내용을 바탕으로 답변을 생성
- 폐기물 사진을 찍거나 갤러리에서 등록 시 폐기물을 인식하여 폐기물 분류와 분리배출 방법을 띄워주는 기능

### 소셜 로그인/회원가입 기능
- 구글, 카카오 계정을 이용하여 로그인 및 회원가입

### 마이페이지 기능
- 사용자가 작성한 게시물 표시
- 사용자가 작성한 댓글 표시
